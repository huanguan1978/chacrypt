## Наша экосистема инструментов, ориентированная на конфиденциальность

Все наши инструменты разработаны с обязательством быть кроссплатформенными, без рекламы, без регистрации, чистыми и бесплатными. Мы ставим вашу конфиденциальность превыше всего, обеспечивая профессиональный и полностью свободный пользовательский опыт.

### Инструменты конфиденциальности и шифрования
- **ChaBox**: Комплексный набор инструментов с фокусом на конфиденциальность для безопасного управления файлами. [GitHub](https://github.com/huanguan1978/chacrypt/tree/main/chabox)
- **Chapose**: Профессиональная утилита командной строки для надежных и безопасных криптографических операций. [GitHub](https://github.com/huanguan1978/chacrypt/tree/main/chapose)

### Управление файлами и автоматизация
- **ft (filetools)**: Высокопроизводительный CLI для продвинутого управления файлами. Включает точную фильтрацию glob, вторичную фильтрацию по нескольким ограничениям (размер/время/mime), мощный набор встроенных подкоманд и бесшовную оркестрацию Shell/YAML для масштабируемой автоматизации. [GitHub](https://github.com/huanguan1978/ft)
- **FileShows**: Визуальный компаньон для `ft`. Помимо интуитивно понятных GUI-операций и визуальной пакетной обработки, он поддерживает автоматизированное планирование для ваших пользовательских наборов команд YAML и комплексное интегрированное управление логами выполнения. [Web](https://webpath.iche2.com/fssdoc/en/)

### Удобный инструментарий генеративного ИИ для начинающих
- **Gai**: Инструменты генеративного ИИ, разработанные для начинающих. Благодаря чистой архитектуре без предустановок, полному контролю параметров и подробной встроенной документации, вы сможете уверенно освоить ключевые технологии ИИ через практический опыт, позволяя ИИ по-настоящему служить вашей жизни и работе. [Web](https://webpath.iche2.com/gaidoc/en/)
- **PyGai**: Мощный мост автоматизации на Python для `Gai`, позволяющий пакетную вставку промптов, интеллектуальное планирование контента и бесшовную интеграцию рабочих процессов ИИ. [GitHub](https://github.com/huanguan1978/pygai)
